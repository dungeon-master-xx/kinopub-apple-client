# Введение

Прежде чем осуществлять запросы к API требуется пройти аутентификацию. Получив access_token к каждому запросу API требуется добавлять GET параметр access_token=<полученный токен> Для всех запросов желательно включить cookie, в них передается csrf токен, без него POST запросы не будут проходить проверку.  

**Параметры запросов заключенные в::**

-

**[]** - необязательные
-

**<>** - обязательные
